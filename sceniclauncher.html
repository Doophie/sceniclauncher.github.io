I'm too lazy to write proper documentation right now, so heres a list of possibly helpful tips!

To open settings, swipe down to access the all applications list, scroll all the way to the bottom. Your settings icon will be shown at the end of the list

To export a layout, open it in the settings and find the export button at the top of the page.

Exported layouts will be a '.slayout' file. The file is actually just a zip with a JSON config and a bunch of images, you can create custom layouts by defining an equivalent JSON and including your own images. Currently I don't have the JSON format documented so you will need to export a layout to see what kind of format it should look like.
